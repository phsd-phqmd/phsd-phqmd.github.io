# phsd-phqmd.github.io

Website for the **PHSD / PHQMD** projects.

- **Live site:** https://phsd-phqmd.github.io/
- **Theme:** BeautifulHugo (Hugo)

---

## Project structure (Hugo)

- `content/` — pages (Markdown)
- `static/` — images, files, favicon (served from `/`)
- `layouts/` — custom templates/partials/shortcodes (overrides theme)
- `themes/beautifulhugo/` — theme submodule / theme files
- `public/` — generated site (output)

> You should edit **content/**, **static/**, and **layouts/**.  
> The `public/` folder is generated by Hugo.

---
## Building locally

To work locally with this project, you'll have to follow the steps below:

1. Fork, clone or download this project
1. [Install][] Hugo
1. Preview your project: `hugo server`
1. Add content
1. Generate the website: `hugo` (optional)

Read more at Hugo's [documentation][].

### Preview your site

If you clone or download this project to your local computer and run `hugo server`,
your site can be accessed under `localhost:1313/hugo/`.

[install]: https://gohugo.io/overview/installing/
